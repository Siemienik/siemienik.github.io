# Installation

`Xlsx-renderer` is published on [`npmjs.com`](https://www.npmjs.com/package/xlsx-renderer)

## How to install `xlsx-renderer`?

### Npm:

```bash
npm install --save xlsx-renderer
```

### Yarn:

```bash
yarn add xlsx-renderer
```
